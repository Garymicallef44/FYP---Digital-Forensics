# Content-Based 2D Image Provenance Analysis: A Digital Forensics Approach
This is a repository containing my submission for a content-based image provenance analysis system.

## Installation

Install Python 3, then install the following dependencies:

```bash
pip install opencv-python numpy pillow networkx
```

if the opencv installation doesn't work try:
```bash
pip install opencv-contrib-python
```

## How to Run
To run the application, execute the following command in your terminal:

```bash
python ProvenanceMenu.py
```
Two test paths are provided in the project directory, they may be used to test the functionality of the system.
